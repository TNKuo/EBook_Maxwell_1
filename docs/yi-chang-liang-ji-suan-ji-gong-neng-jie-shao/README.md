# 一、場量計算機功能介紹

## 六大區塊

場量計算機的介面主要分為六大區塊(如圖1-1)：場量資料區、場量資料管理處理、場量計算顯示區、專案設計變量管理區、暫存命令區、計算機功能按鍵區。

<figure><img src="../assets/image (36).png" alt=""><figcaption><p>圖1-1</p></figcaption></figure>

其中場量資料區，主要是顯示系統和使用者自訂的物理場量名稱及其運算式，並且可以通過右側的按鈕刪除使用者自訂的物理場量。

場量資料管理區：可以將暫存器頂層的物理場量輸出到場量管理器，或將選中的場量輸入到棧寄存器頂層。將使用者自訂的物理場量以腳本形式輸出，或以腳本形式導入。

場量計算顯示區：場計算器採用暫存器的形式來管理輸入輸出。位於面板中間的堆疊暫存器就相當於計算器的顯示器，用於顯示計算器的操作結果。

專案、設計、變量管理區：由於場處理器只能對瞬態場某一個時刻，或渦流場某一個頻率，或參數化分析中某一種有場保存資料的某一種參數組合進行操作。此處可以設置所處理場的時刻、頻率、參數組合等參數。
