# Core Loss

## Core Loss Calculation

• 磁芯材料通常是用於電力變壓器、感應馬達、發電機和其他類似設備中，用於增強磁場或儲存磁能。

磁芯材料在交流磁場中會發生磁滯現象和涡流損耗，這些都會導致磁能轉化為熱能。磁滯是指磁場方向變化時磁芯材料內部的磁化強度不能瞬間改變的現象，造成磁能的能量損失。涡流是指當磁芯材料處於交流磁場中時，磁場的變化會在材料內部產生電流，這些電流導致材料產生能量損失。

• 磁芯損耗的大小取決於磁芯材料的特性、交流磁場的頻率和強度、以及磁芯材料的尺寸和形狀等因素。為了減少磁芯損耗，可以使用高品質的磁芯材料，控制交流磁場的頻率和強度，以及設計適合的磁芯結構。

• Core Loss 是Maxwell已經既有的場量，一樣可以利用場量計算機做計算。
