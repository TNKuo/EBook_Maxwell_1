# Field Calculator of Maxwell

## AEDT物理量

Maxwell電磁場求解的本質，是用數值解法求解給定邊界條件下的Maxwell Equation。以及由方程組演化出的其他偏微分方程問題，而這些偏微分方程的未知量往往是一些最基本的物理場量，隨著求解器的不同求解的基礎物理量也不相同。例如電場一般求解的是純量電勢，二維磁場求解的是向量磁位等。(<mark style="color:orange;">註 1</mark>)

實際研究和工程中，使用者需要關注的物理量，遠遠不止偏微分方程解出的這幾個非常有限的基礎物理場量。這時候，就需要對基礎物理場量做一些向量代數計算和幾何運算，推導出其他的物理場量，來擴展輸出。

在ANSYS的軟件中，例如HFSS、Maxwell等的工程分析軟體一般都會在基礎物理常量以外，定義很多其他常用輸出，例如B、H、J、損耗、電壓電流、電感、力和轉矩等，這些輸出能夠滿足絕大部分工程的需求。但還是有一些不常用的物理常量，不在軟體的預設輸出中，這時候就需要軟體有使用者自訂物理常量輸出的功能，來補充默認輸出的不足。Maxwell的場計算器，就是一個嵌套在Maxwell軟體內部一個這樣的工具。

## Field Calculator

在眾多電磁場軟件中，場處理器是Maxwell非常有特色的一個功能，使用者需要的物理常量，只要是可以基於基礎物理常量，根據物理關係和數學運算式推導出，就可以在場處理器中定義，並在Maxwell中輸出，包括曲線輸出和場圖顯示、參數化和優化分析等。可以說，有了場處理器，Maxwell既可以用作通用的電磁場計算工具，也可以用於專業的科學研究利器。
