# Magnetic Flux

## Magnetic Flux for Magnetostatic

• 磁通量，簡單來說是磁場通流通過特定平面的量度，其大小取決於磁場的強度以及平面的面積和磁場與平面法向量的夾角。可用場量計算機，算出磁通量。

• 磁通量通常用希臘字母Φ（Phi）表示，其單位是韋伯（Weber），代表磁場經過一平方米的面積時產生的磁通量。如果磁場是通過垂直於面積的表面，磁通量可以用下列公式來計算：

Φ = B x A

其中，Φ表示磁通量，B表示磁場強度，A表示平面的面積， x 代表乘法。如果磁場不垂直於面積，那麼公式中還需要乘上cos(θ)，其中θ表示磁場與法向量的夾角。

• 磁通量是一個非常重要的概念，尤其是在磁學、電學、電機等領域。例如，在電動機中，磁通量可以影響轉子產生的電動勢，從而影響電動機的性能。在變壓器中，磁通量的改變可以導致電壓的變化。因此，對磁通量的測量和控制是這些領域中非常重要的技術。

## 什麼時候觀察B場呢？什麼時候觀察H場呢?

讀者可以思考一下，用Maxwell分析磁性元件，例如無線充電或變壓器，什麼時候觀察B場呢？什麼時候觀察H場呢?

下面是我的建議：

### 🔹 什麼時候觀察 **B場（磁通密度，Tesla）**？

B場是最直接與磁性材料（core）內部**磁飽和、磁通路**有關的物理量：

#### ✅ 適用時機：

1. **檢查磁心是否飽和**\
   → 若B值超過材料的飽和點，會導致電感下降與損耗增加。
2. **觀察磁通分佈與磁通路設計**\
   → 用來確認磁通是否均勻、是否集中在預期區域（避免漏磁）。
3. **計算磁通量**\
   → 利用 Φ=∫B⋅dA，求出線圈鏈結的總磁通。
4. **確認變壓器或無線充電線圈的耦合程度**\
   → 觀察磁通是否順利從一次側導引到二次側。

***

### 🔹 什麼時候觀察 **H場（磁場強度，A/m）**？

H場與激磁有關，是由電流產生的磁場強度，與材料特性（例如μ）無關：

#### ✅ 適用時機：

1. **觀察繞線或空氣中磁場分佈**\
   → 可用於檢查外部磁場影響、EMI問題或感應加熱區域。
2. **計算磁導與磁阻（Reluctance）**\
   → 利用 MMF=∫H⋅dl幫助建立磁路模型。
3. **比較不同材料的磁導特性**\
   → 觀察同樣激磁下不同材料對磁通的影響。
4. **繞線區域或空氣中的磁壓降分析**\
   → 特別在磁路分析（磁阻模型）或共模/漏磁路徑設計上重要。

| 項目           | B場（磁通密度）                     | H場（磁場強度）   |
| ------------ | ---------------------------- | ---------- |
| 單位           | Tesla (T)                    | A/m        |
| 與材料有關？       | ✅ 是，取決於材料的μ                  | 🔶無，與材料無關  |
| 用來觀察磁飽和？     | ✅ 適用                         | 不適用        |
| 空氣中磁場分析？     | 🔶特殊情況使用，例如Current Sensor設計。 | ✅ 適用       |
| 用來計算磁通？      | ✅ Φ=∫B⋅dA                    | ❌          |
| 適用於EMI、加熱分析？ | 一般                           | ✅ 適合分析外部磁場 |
